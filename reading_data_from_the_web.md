## Learning Assessment

``` r
books_html=
  read_html("https://books.toscrape.com")

title_vec=
  books_html |> 
  html_elements("h3") |> 
  html_text()

stars_vec=
  books_html |> 
  html_elements(".star-rating") |> 
  html_attr("class")

prices_vec=
  books_html |> 
  html_elements(".price_color") |> 
  html_text()

books_df=
  tibble(
    title = title_vec,
    stars = stars_vec,
    price = prices_vec
  )
```
